# 工业平台项目

## 项目简介
这是一个工业品交易平台，提供商品展示、询价、用户管理等功能。

## 功能特性
- 用户注册与登录
- 商品信息管理
- 在线询价系统
- 响应式网页设计

## 技术栈
- 后端：Node.js, Express, MongoDB, Mongoose
- 前端：HTML, CSS, JavaScript, EJS
- 认证：JWT
- 密码加密：bcryptjs

## 安装与运行

### 前提条件
- Node.js (v14+)
- MongoDB
